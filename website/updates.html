Apple’s iOS offers consistent and timely software updates and security patches. If you want the same experience on Android, then you must buy one of Google’s Pixel phones. This is how iOS version shares break down according to the official Apple Developer website:

  • iOS 12: 23%
  • Earlier: 7%
iOS 13: 70% Around 70% of all iOS devices are now running the latest version. That figure will be a lot lower for Android. While Android 11 is already open for public beta testing, only about 8% of Android devices are running the latest, Android 10. This is how Android breaks down according to the stats available in Android Studio, and is accurate up to May 2020.
  • Android 10: 8.2%
  • Android 9.0 Pie: 31.3%
  • Android 8.1 Oreo: 14%
  • Android 8.0 Oreo: 7.3%
  • Android 7.1.1 Nougat: 5.4%
  • Android 7.0 Nougat: 7.5%
  • Older: 1.9%
If you want the latest features, bug fixes, and security updates, then you should choose iOS.
